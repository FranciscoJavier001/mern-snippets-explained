# mern-snippets-explained

Mini repo para **leer y explicar** snippets reales del stack MERN.
Tu objetivo: abrir cada archivo y **agregar tus propias notas** (qué hace, conceptos, dudas).

## Cómo correr el backend
```bash
cd backend
npm init -y
npm i express
node express-basic.js
```

Luego visita:
- http://localhost:3000/health
- http://localhost:3000/hello/Francisco
